# Speed App - Test Velocità Internet

Applicazione moderna per testare la velocità della connessione internet con interfaccia grafica.

![SpeedApp Icon](icon.ico)

## 📚 Librerie Utilizzate

- **customtkinter** - Framework per interfaccia grafica moderna e responsiva
- **speedtest-cli** - Libreria per eseguire test di velocità della connessione internet
- **threading** - Gestione operazioni asincrone per mantenere l'UI responsive

## 🚀 Cosa Fa l'Applicazione

SpeedApp è un'applicazione desktop che permette di misurare la velocità della tua connessione internet in modo semplice e intuitivo. L'app esegue test di:

- **Ping** - Misura la latenza della connessione
- **Download** - Velocità di download in Mbps
- **Upload** - Velocità di upload in Mbps

### Caratteristiche Principali

✨ Interfaccia grafica moderna e intuitiva  
🌓 Supporto tema chiaro/scuro  
⚡ Test velocità in tempo reale con animazioni  
🎯 Visualizzazione chiara dei risultati  
🛑 Possibilità di interrompere il test in qualsiasi momento  
🔄 Reset rapido per nuovi test
